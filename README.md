🏧 ATM System (Python):-

A simple, menu-driven ATM system built in pure Python.
This project includes:
- 🧩 Procedural (function-based) implementation
- 🏗 Class-based implementation using Atm
- 💾 File handling for data persistence
- 🔐 Account validation and PIN authentication
- 💸 Deposit, Withdraw, Transfer
- 📜 Transaction history (last 5 records)

▶️ Run:-
- python atm.py
- python main.py

- Requires Python 3.

📁 Notes:-

- Account number: 10 digits
- PIN: 4 digits
- Data stored in atm.txt (ignored in .gitignore)
