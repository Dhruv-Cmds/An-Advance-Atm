# An Advance ATM

## Features
- User Authentication
- Balance Inquiry
- Cash Withdrawal
- Deposit Funds
- Transaction History

---

1. Clone the repository:
   ```bash
   git clone https://github.com/Dhruv-Cmds/An-Advance-Atm.git
   ```
   
2. Navigate into the project directory:
  ```bash
  cd An-Advance-Atm
  ```
3. Run the program:
  python main.py

## Validation Rules
- User balance must not go below zero after withdrawal.
- All transactions must be logged for auditing purposes.
